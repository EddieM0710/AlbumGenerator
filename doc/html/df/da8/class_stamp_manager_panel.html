Detailed Description

StampManagerPanel is the main window class for the application, It contains a splitter window with the StampTreeCtrl on one side and a wxNotebook on the other with the the DescriptionPanel and wxScrolledWindow m_galleryPage as pages.

m_galleryPage manages a list of GalleryImagePanel images. StampManagerPanel inherits from wxPanel.
